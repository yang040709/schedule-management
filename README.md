# 简程

一个简洁的日程管理网站

特点：

1. 简洁，非常方便就可以管理您的行程
2. 安全，纯前端，不需要担心数据泄露
3. 多端适配，支持PC、手机、平板等多种设备
4. 能够通过流程图管理日程
5. 支持导入和导出日程

技术栈使用：

1. vue3
2. tailwindcss
3. shadcn-vue-next
4. vue-use
5. vue-flow
6. vee-validate
7. lucide-vue-next
