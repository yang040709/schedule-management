# 简程

一个现代的、简洁易用的日程管理网站，帮助您高效管理日常行程和任务。

## 项目特色

- **简洁易用**：直观的界面设计，快速上手，轻松管理您的行程
- **数据同步**：数据会通过云端同步，确保在不同设备上的一致性
- **多端适配**：完美支持PC、手机、平板等多种设备
- **流程图管理**：创新的流程图式日程规划，可视化编排任务流程
- **统计分析**：提供日程统计功能，帮助您了解自己的日程习惯的情况
- **数据便携**：支持日程数据的导入和导出，方便数据迁移

## 技术栈

### 前端框架

- **Vue 3** - 现代化前端框架
- **TypeScript** - 类型安全的JavaScript
- **Vue Router** - 路由管理

### UI组件与样式

- **Tailwind CSS** - 实用优先的CSS框架
- **shadcn-vue-next** - 现代化UI组件库
- **Lucide Vue Next** - 精美图标库

### 核心功能库

- **Vue Flow** - 流程图和节点编辑器
- **Vee-Validate** - 表单验证
- **Pinia** - 状态管理
- **VueUse** - Vue组合式工具集

### 开发工具

- **Vite** - 快速构建工具
- **Vitest** - 单元测试框架
- **Prettier** - 代码格式化

## 快速开始

### 环境要求

- Node.js 20.19.0 或更高版本
- pnpm 包管理器

### 安装依赖

```bash
pnpm install
```

### 开发模式运行

```bash
pnpm dev
```

应用将在 http://localhost:5173 启动

### 构建生产版本

```bash
pnpm build
```

### 运行测试

```bash
pnpm test:unit
```

## 项目结构

```
src/
├── components/          # 可复用组件
│   ├── ui/             # 基础UI组件
│   ├── Schedule/       # 日程相关组件
│   ├── Flow/           # 流程图组件
│   └── Model/          # 模态框组件
├── views/              # 页面组件
│   ├── Today.vue       # 今日日程
│   ├── Calendar.vue    # 日历视图
│   ├── Flow.vue        # 流程图视图
│   └── Setting.vue     # 设置页面
├── stores/             # 状态管理
├── hooks/              # 自定义组合式函数
├── utils/              # 工具函数
├── types/              # TypeScript类型定义
└── api/                # API接口
```
